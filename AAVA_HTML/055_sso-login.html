SSO Login

Thecomponent is a comprehensive authentication interface that provides both traditional username/password login and Single Sign-On (SSO) capabilities. It features form validation, multiple size variants, loading states, and comprehensive event handling for building secure authentication systems, user portals, and enterprise login interfaces.

<aava-sso-login>

A complete SSO login form with username/password fields, keep signed in option, and SSO login button.
